# Documentación técnica y operativa

Esta carpeta contiene un ejemplo realista de documentación para una aplicación de monitorización energética de plantas fotovoltaicas basada en adquisición IEC60870-5-102, estaciones meteorológicas Modbus TCP, historización, alarmas y visualización SCADA.

## Estructura

- `architecture/` → arquitectura funcional, componentes, datos y decisiones de diseño
- `manuals/` → manual de usuario y manual técnico
- `operations/` → operación diaria, monitorización y troubleshooting
- `api/` → contratos y criterios de integración
- `deployment/` → instalación, configuración y puesta en marcha
- `maintenance/` → mantenimiento preventivo y correctivo
- `security/` → controles de acceso y criterios de endurecimiento

## Objetivo

El objetivo de esta documentación es que un equipo técnico tercero pueda:

- entender la solución
- desplegarla
- operarla
- mantenerla
- ampliarla sin depender del desarrollador original
