# Baseline de seguridad

## 1. Acceso

- cuentas nominales
- contraseñas robustas
- separación de roles
- mínimo privilegio

## 2. Servidor

- actualizaciones planificadas
- firewall restringido
- acceso remoto controlado
- registros de auditoría habilitados

## 3. Aplicación

- validación de entrada en API
- control de acceso a recursos sensibles
- protección de credenciales en configuración
- no exponer directamente la base de datos a terceros

## 4. Operación

- revisión periódica de usuarios
- revocación de accesos obsoletos
- backup cifrado si aplica
- almacenamiento de documentación en repositorio controlado
