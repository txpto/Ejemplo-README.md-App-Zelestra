# Checklist de puesta en marcha

## Antes de FAT/SAT

- [ ] servidor instalado y accesible
- [ ] base de datos operativa
- [ ] ficheros CSV validados
- [ ] IP y puertos confirmados
- [ ] contadores accesibles
- [ ] estaciones meteo accesibles
- [ ] usuarios creados
- [ ] roles definidos

## Durante la prueba

- [ ] comprobación de estados de comunicación
- [ ] comprobación de valores instantáneos
- [ ] comprobación de históricos
- [ ] comprobación de alarmas
- [ ] comprobación de navegación entre pantallas
- [ ] comprobación de exportación/integración

## Cierre

- [ ] acta de aceptación
- [ ] documentación entregada
- [ ] backups iniciales verificados
- [ ] soporte post-puesta en marcha planificado
