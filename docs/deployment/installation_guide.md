# Guía de instalación

## 1. Requisitos mínimos

- servidor Linux o Windows Server
- acceso a base de datos
- acceso IP a equipos de campo
- almacenamiento suficiente para históricos
- mecanismo de backup

## 2. Pasos de instalación

1. desplegar aplicación
2. configurar `system.yaml`
3. cargar CSV de catálogo
4. crear o migrar esquema de base de datos
5. arrancar servicios
6. validar conectividad
7. validar inserción de históricos
8. validar visualización

## 3. Validaciones finales

- todos los contadores esperados visibles
- estaciones meteo visibles
- histórico creciendo
- alarmas funcionales
- exportación/API operativa
