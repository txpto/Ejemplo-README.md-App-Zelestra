# Manual de usuario básico

## 1. Objetivo

Este manual describe el uso diario de la plataforma por parte de operadores, responsables de planta y personal de mantenimiento.

## 2. Acceso

El acceso se realiza mediante credenciales nominales. Dependiendo del rol, el usuario podrá:

- visualizar dashboards
- consultar históricos
- revisar alarmas
- exportar información
- acceder a pantallas de detalle

## 3. Pantalla general

La pantalla general muestra:

- mapa o distribución de contadores
- estado de comunicaciones
- número de alarmas activas
- acceso directo al detalle de cada equipo

### Estados típicos
- verde: comunicación correcta
- amarillo: recuperación de datos en curso
- gris: sin datos o equipo no disponible
- rojo: alarma activa o fallo persistente

## 4. Pantalla de detalle de contador

La pantalla de detalle permite consultar:

- valores instantáneos por fase
- energía activa y reactiva
- potencia total y factor de potencia
- estado de comunicación
- barra de progreso de recuperación de datos
- curva histórica reciente

## 5. Consulta de históricos

El usuario puede:

- seleccionar rango temporal
- seleccionar equipo
- seleccionar variable
- exportar resultados si el rol lo permite

## 6. Gestión básica de alarmas

Las alarmas muestran:

- fecha y hora
- equipo afectado
- severidad
- descripción

El usuario puede filtrar por:
- estado
- severidad
- proyecto
- rango temporal

## 7. Buenas prácticas

- no usar cuentas compartidas
- revisar alarmas críticas al inicio del turno
- confirmar conectividad antes de escalar una incidencia
- documentar cualquier intervención manual sobre campo
