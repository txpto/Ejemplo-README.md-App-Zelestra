# Gestión de cambios

## 1. Alcance

Aplica a cambios en:

- CSV de inventario
- reglas de historización
- configuración global
- versión de aplicación
- esquema de base de datos

## 2. Procedimiento recomendado

1. documentar el cambio
2. evaluar impacto
3. probar en entorno no productivo
4. programar ventana si aplica
5. ejecutar cambio
6. validar resultado
7. registrar evidencias

## 3. Evidencias mínimas

- fichero antes y después
- motivo del cambio
- fecha y responsable
- validación posterior
